About This Tool

This tool stamps a text watermark — CONFIDENTIAL, DRAFT, COPY, a client name, anything up to 60 characters — across every page of a PDF. You see a live preview on page 1 as you adjust the text, angle, colour, and opacity, so there's no guessing how it will look. Processing is entirely in-browser; the document never leaves your device, which matters precisely because watermarked documents are usually the sensitive ones.

Practical Uses

  • DRAFT / FOR REVIEW: mark working versions so they can't be mistaken for finals in an email chain.
  • COPY / DUPLICATE: when sharing copies of certificates or agreements while retaining the clean original.
  • Client or recipient names: a light watermark like "Shared with Fernandez & Co" discourages onward forwarding of confidential documents.

Good to Know

The watermark is drawn on top of the page content as part of the PDF itself, so it prints and displays everywhere. Keep opacity around 20–30% so underlying text stays readable. Note that a determined person can remove a text watermark with editing software — treat it as a strong label, not a security control. For documents that truly must not leak, limit distribution rather than relying on markings. Pair with page numbers for court and filing bundles.
