About This Tool

This tool changes the physical page size of a PDF — converting odd-sized documents to standard A4, Letter, or Legal, or scaling the whole document up or down by percentage. Content is scaled proportionally and centered, never cropped or distorted, and landscape pages automatically stay landscape. All processing happens in your browser.

When Page Size Matters

  • Printing: a PDF built at a non-standard size prints with unexpected margins or gets shrunk unpredictably; converting to A4 first gives consistent output.
  • Merging mixed sources: combining an A4 letter with a Letter-size statement and a receipt-sized scan produces a bundle where every page is a different size — normalise everything to A4 first, then merge, and the result looks professional.
  • Portals with size expectations: some e-filing systems assume A4; oversized pages can render zoomed-out or fail validation.

Resizing vs Compressing

Resizing changes the page dimensions; it barely changes the file's size in megabytes. To shrink the download size of a heavy scan, use the compressor — or both: resize to A4 for consistency, then compress to meet the upload limit.
