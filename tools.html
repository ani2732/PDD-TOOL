Free, Private PDF Tools

Every tool on this site follows the same principle: your files never leave your device. Instead of uploading documents to a server for processing, these tools run entirely inside your web browser using open-source JavaScript libraries. That makes them safe for the documents people actually need to work with — bank statements, contracts, identity documents, tax filings, and medical records.

The tools are designed to work together, and every one shows a live preview before anything is changed. A typical workflow: convert phone photos to PDF, fix sideways pages, drop the blanks, merge into one file, sign it, number the pages, and compress to meet an upload limit. For extracting data, bank statements go to Excel and documents go to Word — all without the file leaving your machine.

No signup, no watermarks on your output, no file limits beyond what your device's memory can handle. The site is supported by unobtrusive advertising, which keeps everything free.
