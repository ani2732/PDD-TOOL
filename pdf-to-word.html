About This Tool

This converter rebuilds your PDF as a genuine, editable Word document — and in Formatted mode it preserves far more than words: the original page size and orientation, each line's font size in points, the font class (serif → Times New Roman, sans-serif → Arial, monospace → Courier New), left / centre / right alignment detected from each line's position, paragraph spacing from the vertical gaps, and — most usefully — tables reconstructed as real Word tables with borders and proportional column widths, using positional column detection. A summary above the preview tells you exactly what was detected before you download. Everything runs in your browser; nothing is uploaded.

What Formatted Mode Preserves — and What It Can't

  • Preserved: page dimensions, font sizes, font families (by class), alignment, blank-line spacing, page breaks, and tabular data as editable Word tables.
  • Approximated: exact typefaces (PDFs embed fonts Word may not have — the nearest standard family is used) and complex multi-column magazine layouts, which convert in reading order.
  • Not carried over: images, logos, background graphics, and colours — extracting those faithfully is what desktop conversion engines exist for. If the visual appearance itself is what you need, keep the PDF or use PDF to JPG; Formatted mode is for when you need the document back in editable form.

Scans Need OCR First

A scanned document is a photograph — there is no text inside to extract, and this tool will say so rather than produce an empty file. Run scans through OCR (many scanner apps include it) to add a text layer, then convert here. For tabular data specifically — bank statements, price lists — PDF to Excel gives you the columns as a spreadsheet instead.
